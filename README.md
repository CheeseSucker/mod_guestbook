mod_guestbook
=============

SilverStripe 3.1 Guestbook module.
Not released yet.

## Features ##

 - Emoticons
 - E-mail protection
 - Moderators can comment on entries
 - Edit and delete entries
 - Pagination
 - Settings:
   * Number of entries per page
   * Toggle emoticons
